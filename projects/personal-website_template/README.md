# Проект сайта-визитки

Это простой пример сайта-визитки, который написан на HTML, CSS и ванильном JS в сборщике Vite.

## Использованные технологии

[![My Skills](https://skillicons.dev/icons?i=js,html,css,vite)](https://skillicons.dev)

---

## Особенности

- Сборка через Vite
- Верстка по принципам БЭМ
- Модульная структура скриптов
- Адаптивная верстка

---

## Функционал

- **Слайдер**  
  В разделе «Что отличает меня от других» реализован слайдер с поддержкой **тач-событий**, который включается на экранах менее **992px**.

- **Модальное окно для изображений**  
  В разделе «Отзывы» изображения можно открыть в увеличенном формате, закрыть по Esc, кнопке "Закрыть" по клику по пространству.
